# Kyros Logger
A distributed logging system with customizable log levels and handlers.
## Requirements:
- Go 1.19+
## Installation:
1. Clone the repository: `git clone https://github.com/user/kyros-logger.git`
2. Navigate to the project directory: `cd kyros-logger`
3. Initialize the Go module: `go mod init kyros-logger`
4. Get dependencies: `go get` 
5. Run the application: `go run main.go`
